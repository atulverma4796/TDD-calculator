# TDD-calculator
This is a TypeScript-String Calculator implemented by the principles of TDD. It supports arithmetic operations-including addition, subtraction, multiplication, and division. It also allows for custom delimiters and catches edge cases, such as division by zero and negative numbers.

Features
Takes multiple inputs such as 'add', 'subtract', 'multiply', 'divide' along with numbers from string.
Acceptable delimiters may vary
Error Handling
Negative Number for Addition.
Division by Zero.
TDD with Jest
Automatic testing

Getting Started
1. Clone
2. npm install
3. npm jest
